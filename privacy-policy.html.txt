  <h1>Privacy Policy for Tabwise</h1>
  <p>Effective Date: February 2026</p>

  <h2>1. Overview</h2>
  <p>
    Tabwise is a browser extension that allows users to analyze and compare content from their open tabs using artificial intelligence.
  </p>

  <h2>2. Information We Process</h2>
  <ul>
    <li>Content from browser tabs selected by the user</li>
    <li>Email address (only if provided during subscription purchase)</li>
    <li>Subscription status information</li>
    <li>Usage counters and saved tab groups (stored locally)</li>
  </ul>

  <h2>3. How We Use Information</h2>
  <ul>
    <li>To generate AI responses from selected tabs</li>
    <li>To verify subscription status</li>
    <li>To enforce usage limits</li>
    <li>To provide Pro features to paid users</li>
  </ul>

  <h2>4. Third-Party Services</h2>
  <p>
    Tabwise uses third-party services to operate:
  </p>
  <ul>
    <li>Groq API – for AI processing (Free tier)</li>
    <li>OpenAI API – for AI processing (Pro/Power tier)</li>
    <li>Lemon Squeezy – for subscription payments</li>
    <li>Supabase – for subscription verification</li>
  </ul>

  <p>
    Tab content is sent to AI providers only when a user actively requests analysis.
  </p>

  <h2>5. Data Storage</h2>
  <p>
    Tabwise stores the following locally in the user's browser:
  </p>
  <ul>
    <li>Email address (if provided)</li>
    <li>Subscription status</li>
    <li>Usage counters</li>
    <li>Saved tab groups</li>
  </ul>

  <p>
    We do not sell, rent, or trade personal information.
  </p>

  <h2>6. Data Security</h2>
  <p>
    Communication with third-party services is encrypted via HTTPS.
  </p>
